# てんちょーおみくじ

朝開いて運試しできる、てんちょー風アレンジのおみくじ。

## オリジナル要素
- 通常の大吉〜大凶ではなく **「てんちょーランク」7段階**
  - 神てんちょー / 超てんちょー / てんちょー / ふつうてんちょー / ちょいちょー / てんちょーじゃない… / まさかの店長代理
- 引くと **てんちょーカットイン**（ストライプスイープ＋画像スライド＋効果音）
- 4つのラッキー：言葉／時間／カラー／店内スポット
- てんちょー語録メッセージ
- 1日1回固定（同じ日は同じ結果が出る = リロード対策）
- Xシェアで自慢
